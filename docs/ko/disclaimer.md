## AI Limitations - Hallucinations and Errors

이 프로젝트는 대규모 언어모델(Claude)을 사용합니다. LLM은 **hallucination**, **사실 오류**, **잘못된 법률 분석**을 실제로 생성할 수 있습니다. 산출물은 다음과 같은 문제를 포함할 수 있습니다.

- **존재하지 않는 법적 근거를 꾸며냄**: 없는 법령, 규정, 판례, 행정지침을 인용
- **법 내용을 잘못 설명함**: 이미 개정되었거나 폐지된 규범을 현재 유효한 것처럼 제시
- **관할을 혼동함**: 특정 법 규칙을 다른 국가, 주, 규제기관의 것으로 잘못 귀속
- **중요 쟁점을 누락함**: 관련 규정, 예외, 계류 중인 입법 변화를 놓침
- **출처 위계를 잘못 분류함**: 비구속적 가이드, 블로그, 2차 요약을 1차 법률 자료처럼 취급
- **인용을 날조함**: 그럴듯하지만 실제로는 존재하지 않는 사건명, 사건번호, 조문 번호, 논문 제목을 생성

에이전트에는 Step 4 fact-checking, source grading, quality gate, conflict detection, `[Unverified]` 태그 같은 보호장치가 있지만, **이 장치들은 오류 위험을 낮출 뿐 완전히 제거하지는 못합니다**. 모든 산출물은 반드시 자격 있는 사람이 직접 검토하고 검증해야 합니다. 에이전트는 리서치를 보조하는 도구이며, 최종 판단은 항상 사람이 합니다.

## Data Handling

이 에이전트는 LLM API 호출을 사용하므로, 리서치 질의와 수집된 소스 텍스트가 모델 제공자(Anthropic)에게 전송됩니다. 표준 상업 이용 약관에 따라, API 요청 처리 범위를 넘어 별도 외부 서버에 저장되지는 않습니다.

기밀 정보나 비밀유지 특권이 적용될 수 있는 사안을 이 도구로 다룰 경우, 다음을 확인하시기 바랍니다.

- 적절한 데이터 처리 조건이 포함된 상업용 API 키를 사용할 것
- 최신 조건은 [Anthropic data usage policies](https://www.anthropic.com/policies)에서 검토할 것
- 민감 정보 처리 전에 소속 조직의 보안 및 컴플라이언스 요구사항을 확인할 것
